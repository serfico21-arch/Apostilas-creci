Inferência & Estatística

Base teórica para compreender como dados amostrais permitem conclusões sobre uma população — aplicado à avaliação de imóveis.

conceito 01
Inferência
Raciocínio desenvolvido a partir de indícios. Processo intelectual pelo qual se chega a uma conclusão a partir de premissas — proposições ou enunciados de problemas a resolver.
conceito 02
Estatística
Ramo das ciências matemáticas aplicadas que usa o princípio da teoria das probabilidades no estudo de séries de dados numéricos.
conceito 03
Inferência Estatística
Usa dados de uma amostra para tirar conclusões sobre a população. Na avaliação imobiliária: usa imóveis referenciais para estimar o valor do imóvel avaliando.
// amostra & população

Definições Essenciais

  • POPULAÇÃO: Conjunto de elementos com pelo menos uma característica em comum. Ex.: todos os imóveis residenciais de 2 dormitórios de uma cidade.
  • AMOSTRA: Subconjunto de uma população. Na avaliação imobiliária, é a lista de imóveis referenciais usados para determinar o valor do imóvel avaliando.
// medidas de tendência central

Onde os dados se concentram

x̄

Média

Soma de todos os valores dividida pela quantidade de elementos. Sensível a valores extremos (outliers).

Md

Mediana

Valor central que divide o conjunto ordenado ao meio. 50% dos dados ficam acima, 50% abaixo.

Mo

Moda

Valor que aparece com maior frequência no conjunto de dados. Um conjunto pode ter zero, uma ou várias modas.

📊
Exemplo com 10 imóveis referenciais: R$ 250k, 255k, 268k, 275k, 275k, 280k, 280k, 280k, 287k, 300k → Média: R$ 275.000 · Moda: R$ 280.000 · Mediana: R$ 277.500
// excel · fórmulas

Funções no Excel

Mediana=MED(intervalo)
Moda=MODO.ÚNICO(intervalo)

// estimadores de dispersão

Variabilidade dos Dados

Dispersão mostra o quão espalhados ou concentrados estão os dados em relação à média. Fundamental para interpretar a qualidade de uma amostra.

Amplitude

Distância numérica entre o menor e o maior valor do conjunto. Medida mais simples, sensível a extremos.

Amplitude = Máx − Mín

Variância (s²)

Indica o quão distante cada valor está da média. Calcula a soma dos quadrados dos desvios dividida por (n−1).

s² = Σ(Xᵢ − X̄)² / (n−1)

Desvio Padrão (s)

Raiz quadrada da variância. Indica a regularidade do conjunto em relação à média. Na mesma unidade dos dados.

s = √s²

Distribuição Normal

Distribuição de probabilidade que mostra o percentual de chance de um dado aleatório estar dentro de um intervalo. Base para o Intervalo de Confiança.

// curva normal · regra empírica (68-95-99,72)
± 1 desvio padrão 68,26%
± 2 desvios padrão 95,44%
± 1,96 σ (exato) 95,00%
± 3 desvios padrão 99,72%
🏠
Avaliar um imóvel não é um dado probabilístico diretamente — mas o conceito de distribuição normal é usado para determinar o Intervalo de Confiança do valor apurado na avaliação.
// modelagem

Variável Dependente & Independentes

Na avaliação imobiliária, o preço (variável dependente) é explicado por características do imóvel (variáveis independentes).

⚙️
Variável dependente — é afetada pelas independentes → PREÇO
Variáveis independentes — afetam o preço → m², dormitórios, vagas de garagem, banheiros...
// tamanho da amostra

Fórmula para Quantidade Mínima de Referenciais

fórmula
n = 3 × (k + 1)
n = quantidade mínima de imóveis referenciais  |  k = quantidade de variáveis independentes
📐
Exemplo: Apartamento com 80m², 2 dormitórios e 1 vaga → k = 3 variáveis independentes → n = 3 × (3+1) = 12 imóveis referenciais mínimos.

Coeficiente de Correlação

Método estatístico que mede e quantifica a relação entre duas variáveis. Resultado entre −1 e 1.

// escala de correlação · pearson (r)
−1 (negativa forte) 0 (nula) +1 (positiva forte)
= 1Relação absoluta
0,9–1Muito forte
0,7–0,9Forte
0,3–0,7Média
0–0,3Fraca
= 0Relação nula

↗ Positiva

Quando uma variável aumenta, a outra também aumenta. Ex.: mais m² → maior preço.

↘ Negativa

Quando uma aumenta, a outra diminui. Ex.: mais distância do centro → menor preço.

→ Nula

Não há relação de correspondência entre as variáveis. Dados da variável são irrelevantes.

// regressão linear múltipla

Modelo de Regressão

Técnica estatística para analisar a relação entre uma variável dependente e múltiplas variáveis independentes. Base do modelo de avaliação imobiliária.

equação do modelo
Ŷ = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ
Ŷ = valor estimado (preço)  |  β₀ = interseção  |  β₁…βₙ = coeficientes  |  X₁…Xₙ = variáveis independentes
01

Predição

Com o padrão de variação da variável dependente, predetermina-se valores para as variáveis independentes.

02

Seleção de Variáveis

Elimina variáveis com pouca ou nenhuma contribuição para explicar o modelo.

03

Estimação de Parâmetros

Usa o modelo para definir ações ou atividades baseadas nos coeficientes calculados.

04

Inferência

Realiza análises para confirmar ou refutar hipóteses sobre as relações entre variáveis.

// interpretando os resultados

As 3 Tabelas de Resultado

1

Estatística de Regressão — R²

O R-Quadrado mede o quanto as variáveis X explicam a variação de Y. Varia de 0 a 1. O R² ajustado penaliza a inclusão de variáveis pouco explicativas — use este como referência principal.

✅
R² ajustado > 0,70 — busque conjuntos de dados com este resultado. Resultado de 0,978 significa que as variáveis explicam 97,8% da variação dos preços.
2

ANOVA — F de Significação

Analysis of Variance. Verifica se pelo menos uma variável X explica a variação de Y. Quanto menor, maior a relação linear entre as variáveis.

✅
F de significação < 0,05 — se for maior, reveja as variáveis da amostra.
3

Coeficientes — Valor-P Individual

O Valor-P de cada variável independente indica sua significância individual. Varia de 0 a 1 — quanto menor, mais a variável contribui para explicar o preço. Solução para valor-P alto: remover a variável com maior valor-P e rodar a regressão novamente.

✅
Valor-P < 0,10 para avaliação imobiliária (idealmente < 0,05). Remova a variável com maior valor-P se ultrapassar 0,10.

Roteiro de 10 Passos

Sequência metodológica completa para realizar uma avaliação imobiliária com inferência estatística, conforme o CRECI-SP.

01

Definir as variáveis independentes

Baseadas nas características do imóvel avaliando: m², dormitórios, vagas, banheiros, etc.

02

Calcular o tamanho mínimo da amostra

Usar a fórmula n = 3(k+1), onde k é o número de variáveis independentes.

03

Obter imóveis referenciais

Com características comparáveis ao imóvel avaliando para compor a amostra.

04

Criar a tabela de variáveis

Variáveis independentes nas colunas + variável dependente (preço) separada.

05

Descartar variáveis com correlação nula

Usar PEARSON ou CORREL no Excel. Remover as que não apresentarem correlação com o preço.

06

Criar o modelo de regressão inicial

Com todas as variáveis independentes restantes. Usar a ferramenta Regressão (Análise de Dados).

07

Descartar variável com maior Valor-P

Se alguma apresentar Valor-P superior a 0,10, removê-la e repetir a regressão.

08

Refinar até obter o modelo ideal

F de significância < 0,01 e Valor-P de cada variável < 0,10. R² ajustado > 0,70.

09

Determinar o valor do imóvel

Substituir os valores do imóvel avaliando na equação do modelo de regressão escolhido.

10

Calcular o Intervalo de Confiança

Limite máximo do Alfa em 0,30 (70% de confiança). Apresentar faixa de variação de valor.
